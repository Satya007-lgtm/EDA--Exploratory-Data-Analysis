>Exploratory Data Analysis (EDA) on Amazon E-commerce Earbuds<

>Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on an Amazon e-commerce dataset related to **earbuds**.
The objective is to analyze product pricing, ratings, reviews, and brand performance to extract meaningful business insights.


>>Objectives
- Understand pricing patterns of earbuds on Amazon.
- Analyze customer ratings and reviews distribution.
- Compare brand-wise performance.
- Identify trends influencing customer preferences.
- Derive insights useful for business and marketing decisions.


>>Dataset Description
The dataset contains information such as:
- Product Name
- Brand
- Price
- Ratings
- Number of Reviews
- Product Category (Earbuds)


>>Tools & Technologies Used
- **Python**
- **Jupyter Notebook**
- **Pandas** – Data manipulation
- **NumPy** – Numerical analysis
- **Matplotlib & Seaborn** – Data visualization


>>EDA Steps Performed
1. Data loading and initial inspection.
2. Data cleaning and handling missing values.
3. Exploratory analysis of price and ratings.
4. Brand-wise comparison.
5. Visualization of trends and patterns.
6. Interpretation of results.

>>Key Insights
- Identified price ranges preferred by customers.
- Observed relationship between price and rating.
- Found top-performing brands based on customer feedback.
- Analyzed customer behavior using review patterns.
